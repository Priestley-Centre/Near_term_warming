Code to produce results for Supplementary Table 2 and Supplementary Figure 4.

'Haustein_AWI_obs.csv'
- 2nd column is GWI of Haustein et al. 2017 (https://www.globalwarmingindex.org/AWI/info_page.html)
- 3rd column is average of 4 observational datasets of GBST (HadCRUT4, Cowtan-Way, GISTEMP version 4, and NOAA-MLOST)
