Additional processed data required to plot Figure 3.

'obs_max_Ny_trends_*.npy' - maximum trends in GSAT for a range of N-year periods in 
                            observational historical record
                            
'gsat_Nytrends_Haus_res_HadOST.npy' - time series of N-year trends in GSAT residuals
                                      for HadOST observations minus the forced IRM
                                      response in Haustein et al. (2019).
